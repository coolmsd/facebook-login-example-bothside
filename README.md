## Install
```sh
$npm install
```

## Run
```sh 
$npm start
```


## check server
```sh  
http://localhost:3000/
```

you need app id and app secret of facebook

1. Client side auth
	- using js sdk of facebook
	- config : /auth/conf.js
2. Server side auth
	- using passport-facebook
	- config : /auth/conf.js
